Grounded or Going? Predicting Flight Delays with Logistic Regression

A passion project

I was originally looking at time series forecasting by trying to predict rent price trends in the most walkable cities in the U.S. However, when I was starting that project a friend mentioned an open dataset published by our regional airport. With a lifelong passion for travel, I was immediately intrigued by the potential to analyze this data.

Exploring the dataset

The dataset contained detailed flight information as scheduled and operation departure date & times, destination cities, airlines, day, month and year and more. With some feature engineering, it is prime material for training a binary classifier to predict flight delays.

I am currently in the midst of preprocessing the raw data to prepare it for modeling. Logistic regression will be the first machine learning algorithm I implement for this binary classification tasks.

As I continue developing the data pipeline and logistic regression model, I look forward to refining my data science skills and providing key insights for the business.

Libraries, techniques and model(s) I am applying for this project:

  • Python libraries used: NumPy, Pandas, Matplotlib, Seaborn and Sklearn.
  • Initial EDA, data visualization and feature engineering,
  • Splitting the dataset for training and testing, and preprocessing the data,
  • Applying logistic regression,
  • Model performance evaluation and hyperparameter tuning.

Current stage of this project: work in progress

As mentioned, I am currently working on this project and have begun the preprocessing steps to apply the first model. The Jupyter notebook I am using is in a draft phase and is filled with notes to myself about things to complete, refine, my workflow and some testing code. If you are interested in learning more about the specifics of this project, you can find it here on my GitHub page.
